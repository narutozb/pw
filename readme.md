# 接单流程


1. 沟通
2. 确定主题
3. 邮寄到工作室
4. 再次确认细节
5. 确定拍摄时间和场地
6. 拍摄
7. 出片


# 拍摄主题

* 日式主题
* 暗黑风
  * 鬼怪风
* 可爱主题
